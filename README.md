# Ashok's Men's Parlour - Premium Grooming Since 1996

A high-end, responsive web application for **Mr. Ashok Men's Parlour**, a premier salon located in Dhule, Maharashtra.

## 🚀 Features

- **Premium UI/UX**: Sophisticated "Charcoal & Gold" theme with modern animations using Framer Motion.
- **WhatsApp-Integrated Booking**: A 4-step booking engine that generates pre-filled WhatsApp messages for appointments.
- **Dynamic Services & Products**: Easily manageable data for salon services, staff, and products.
- **Live Status**: Real-time "Open/Closed" indicator based on shop hours.
- **Responsive Design**: Mobile-first approach for seamless experience across all devices.
- **SEO Optimized**: Meta tags and Open Graph data for better search visibility.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router DOM

## 📁 Project Structure

- `src/components/`: Modular UI components (Hero, Navbar, BookingModal, etc.)
- `src/constants/data.js`: Central data repository for services, staff, and business info.
- `src/pages/`: Page-level components.
- `public/`: Static assets like videos and images.

## 🏁 Getting Started

1.  **Clone the repository**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
4.  **Build for production**:
    ```bash
    npm run build
    ```
